# Laptop Price Prediction System
This project uses Machine Learning (XGBoost) to predict laptop prices based on hardware specifications.

## Files in this Repository:
- `B4_B1_05_SPL_Project.ipynb`: Main Colab Notebook
- `app.py`: Streamlit GUI Code
- `Laptop_Dataset.csv`: Dataset used for training
- `laptop_price_model.pkl` & `scaler.pkl`: Trained Model and Scaler files
